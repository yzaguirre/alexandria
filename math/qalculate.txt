Qalculate! library and CLI
https://github.com/Qalculate/libqalculate
https://qalculate.github.io/manual/qalc.html
	wget https://github.com/Qalculate/libqalculate/releases/download/v5.0.0/qalculate-5.0.0-x86_64.tar.xz
	tar xf qalculate-5.0.0-x86_64.tar.xz
	cd qalculate-5.0.0
	./install
Windows
	winget install Qalculate.Qalculate
	choco install qalculate
Homebrew
	brew install qalculate-gtk
Configuration at
~/.config/qalculate/qalc.cfg
