WinMerge is an Open Source differencing and merging tool for Windows.
WinMerge can compare both folders and files,
presenting differences in a visual text format that is easy to understand and handle.
https://winmerge.org
https://github.com/WinMerge/winmerge
https://manual.winmerge.org/en/
	winget install WinMerge.WinMerge
	choco install winmerge
