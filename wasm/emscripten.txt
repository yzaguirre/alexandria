Emscripten: An LLVM-to-WebAssembly Compiler
https://emscripten.org

Windows
Instead of running emscripten on Windows directly,
you can use the Windows Subsystem for Linux to run it in a Linux environment.
	choco install emscripten
Homebrew
	brew install emscripten

Usage
	emsdk
