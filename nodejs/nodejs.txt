Environment Variables
https://stackoverflow.com/questions/13333221/how-to-change-value-of-process-env-port-in-node-js
For just one run (from the unix shell prompt):
	PORT=1234 node app.js
More permanently:
	export PORT=1234
	node app.js
In Windows:
	set PORT=1234
In Windows PowerShell:
	$env:PORT = 80
