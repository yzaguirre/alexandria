Trunk helps in build and package a web assembly application
	cargo install trunk
Add web assembly as compilation target
	rustup target add wasm32-unknown-unknown
Create the package
	cargo new RustFrontEnd
	cd RustFrontEnd
Serve web app locally
	trunk serve
