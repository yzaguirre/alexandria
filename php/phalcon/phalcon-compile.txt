	# apt-get install php5-dev php5-mysql gcc libpcre3-dev

On some Linux installations, the compilation may fail because  libpcre3-dev
is missing. You can fix this issue by installing this package with your Linux
package manager.

	wget https://github.com/phalcon/cphalcon/archive/phalcon-v2.0.2.tar.gz
	tar xf phalcon-v2.0.2.tar.gz
	cd cphalcon-phalcon-v2.0.2/build
	sudo ./install

2. Add the extension to your 30-phalcon.ini:
https://github.com/phalcon/cphalcon/issues/2224
	echo $'[phalcon]\nextension=phalcon.so' > /etc/php5/mods-available/phalcon.ini
	php5enmod phalcon
