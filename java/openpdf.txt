OpenPDF is a free Java library for creating and editing PDF files with a
LGPL and MPL open source license.
OpenPDF is based on a fork of iText.
https://github.com/LibrePDF/OpenPDF/releases/tag/1.3.0
https://news.ycombinator.com/item?id=20552365

Use OpenPDF as Maven dependency
See maven.txt
	vim pom.xml
<dependency>
  <groupId>com.github.librepdf</groupId>
  <artifactId>openpdf</artifactId>
  <version>1.3.30</version>
</dependency>
