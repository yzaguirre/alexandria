Helps with unknown plugins
Create profile directory
	mkdir ~/.config/chromium/Jail
Start chromium
	chromium --user-data-dir=$HOME/.config/chromium/Jail
Make an alias
	alias chromiumjails='chromium --user-data-dir=$HOME/.config/chromium/Jail'
