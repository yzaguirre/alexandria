ntfy (pronounce: notify) is a simple HTTP-based pub-sub notification service.
It allows you to send notifications to your phone or desktop via scripts from any computer,
entirely without signup, cost or setup.
https://ntfy.sh
Send push notifications to your phone or desktop using PUT/POST
https://github.com/binwiederhier/ntfy
	wget https://github.com/binwiederhier/ntfy/releases/download/v1.28.0/ntfy_1.28.0_linux_amd64.deb
	wget https://github.com/binwiederhier/ntfy/releases/download/v1.28.0/checksums.txt
	sha256sum -c --ignore-missing checksums.txt
	dpkg -i ntfy_1.28.0_linux_amd64.deb
	vim /etc/ntfy/server.yml
	systemctl enable ntfy
	systemctl start ntfy
	mv /usr/bin/ntfy{,-cli}
Windows
	scoop install ntfy

	wget https://github.com/binwiederhier/ntfy/releases/download/v1.28.0/ntfy_1.28.0_windows_x86_64.zip
