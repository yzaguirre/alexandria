Xplr - A hackable, minimal, fast TUI file explorer,
stealing ideas from nnn and fzf.
Written in Rust
https://xplr.dev

Install
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-linux.tar.gz
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-linux.tar.gz.asc
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-linux.sha256
	gpg --keyserver hkps://keyserver.ubuntu.com --recv-keys 0xD59CA14710C17C6B24717AF90F8EF5258DC38077
	gpg --verify xplr-linux.tar.gz.asc
	sha256sum -c xplr-linux.sha256
	sudo tar xf xplr-linux.tar.gz -C /usr/local/bin
MacOS
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-macos.tar.gz
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-macos.tar.gz.asc
	wget https://github.com/sayanarijit/xplr/releases/download/v0.17.4/xplr-macos.sha256
	gpg --verify xplr-macos.tar.gz.asc
	shasum -a 256 -c xplr-macos.sha256
	tar xf xplr-macos.tar.gz -C ~/bin
Homebrew
	brew install xplr

Configuration
See ../documents/yaml.txt
	mkdir -p ~/.config/xplr
	xplr | yq ".config" -y | tee ~/.config/xplr/config.yml
Or
	mkdir ~/.config/xplr/
	cd ~/.config/xplr/
	wget https://github.com/sayanarijit/.files/raw/Linux/nixpkgs/files/xplr/config.yml
