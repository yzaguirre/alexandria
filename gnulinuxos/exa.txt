https://the.exa.website/
https://github.com/ogham/exa
Package install
	apt install exa
Binary install
	wget https://github.com/ogham/exa/releases/download/v0.9.0/exa-linux-x86_64-0.9.0.zip
	wget https://github.com/ogham/exa/releases/download/v0.9.0/MD5SUMS
	wget https://github.com/ogham/exa/releases/download/v0.9.0/SHA1SUMS
	unzip exa-linux-x86_64-0.9.0.zip
	sha1sum -c --ignore-missing SHA1SUMS
	md5sum -c --ignore-missing MD5SUMS
Move binary to a PATH directory (~/bin, /usr/local/bin, other)
	mv exa-linux-x86_64 /usr/local/bin/exa
Useful alias
	alias ll='exa -bghHlS'
Other examples
	exa -R
	exa -T
	exa -R --level=4
	exa -l --sort=size
	exa -lr --sort=size
