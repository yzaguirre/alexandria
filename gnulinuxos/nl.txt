	nl birthdays
Multiple documents
	nl birthdays birthdayssorted
Add different values through options
	nl -i 3 birthdays
Change justification
	nl -n rz birthdays
Change number width
	nl -w 20 birthdays
It will fold if length exceeds width
	nl -w 15 birthdays
