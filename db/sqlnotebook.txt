Casual data exploration in SQL
Open source app for casual data exploration in SQL
Import your data from CSV, Excel, Microsoft SQL Server, PostgreSQL, and MySQL.
Then use a Jupyter-style notebook interface for exploratory queries,
and write stored procedures for reusable logic.
SQL Notebook is powered by an extended SQLite engine,
supporting both standard SQL queries and SQL Notebook-specific commands and functions.
