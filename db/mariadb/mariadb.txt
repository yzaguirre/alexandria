Once the key is imported and the repository added you can install MariaDB with:

	apt update
	apt install mariadb-server


Here are the commands to run to install MariaDB on your Ubuntu system:
	sudo apt-get install software-properties-common
	sudo apt-key adv --recv-keys --keyserver hkp://keyserver.ubuntu.com:80 0xcbcb082a1bb943db
	sudo vim /etc/apt/sources.list.d/mariadb.list
# MariaDB 10.0 repository list - created 2014-06-17 19:35 UTC
# http://mariadb.org/mariadb/repositories/
deb http://sfo1.mirrors.digitalocean.com/mariadb/repo/10.0/ubuntu trusty main
deb-src http://sfo1.mirrors.digitalocean.com/mariadb/repo/10.0/ubuntu trusty main

Once the key is imported and the repository added you can install MariaDB with:
	apt update
	apt install mariadb-server
