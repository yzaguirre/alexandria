Install
https://github.com/mickael-kerjean/filestash/pull/136
	mkdir filestash && cd filestash
	curl -O https://downloads.filestash.app/latest/docker-compose.yml
	docker-compose up -d
Once the installation has completed, open up a browser and go to:
http://your_ip:8334
Enter the admin password you want to use to protect the admin console.
The admin console is available at the /admin url.
