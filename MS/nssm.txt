	choco install NSSM

Usage
https://stackoverflow.com/questions/415409/run-batch-file-as-a-windows-service
Install a service
	nssm install "YourCoolServiceNameLabel"
Uninstall a service
	nssm remove "YourCoolServiceNameLabel"
