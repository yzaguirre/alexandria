Pdfium.Net SDK

NuGet install
	Install-Package Pdfium.Net.SDK
Install specified package in specified project
	Install-Package pdfium.net.sdk - ProjectName <Project Name>
DotNet
	dotnet add package Pdfium.Net.SDK

	7z x -opackages pdfium.zip net60
