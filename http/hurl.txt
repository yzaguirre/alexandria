Hurl, run and test HTTP requests.
https://hurl.dev/index.html
https://github.com/Orange-OpenSource/hurl
https://news.ycombinator.com/item?id=28758226
Install
https://hurl.dev/docs/installation.html
	wget https://github.com/Orange-OpenSource/hurl/releases/download/1.7.0/hurl_1.7.0_amd64.deb
	dpkg -i hurl_1.7.0_amd64.deb
Windows
	winget install Orange-OpenSource.Hurl
Installer
	wget https://github.com/Orange-OpenSource/hurl/releases/download/1.7.0/hurl-1.7.0-win64-installer.exe
	explorer.exe hurl-1.7.0-win64-installer.exe
Homebrew
	brew install hurl

Usage
	hurl --progress --summary --html . test1.hurl

Export hurl file
	hurlfmt test.hurl --format json | jq
	hurlfmt access_file_list.hurl --format json -o access_file_list_POSTMAN.json
