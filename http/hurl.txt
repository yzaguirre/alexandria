Install
https://hurl.dev/docs/installation.html
	wget https://github.com/Orange-OpenSource/hurl/releases/download/1.6.0/hurl_1.6.0_amd64.deb
	dpkg -i hurl_1.6.0_amd64.deb
Windows
	winget install Orange-OpenSource.Hurl
Installer
	wget https://github.com/Orange-OpenSource/hurl/releases/download/1.6.0/hurl-1.6.0-win64-installer.exe
	explorer.exe hurl-1.6.0-win64-installer.exe

Usage
	hurl --progress --summary --html . test1.hurl
